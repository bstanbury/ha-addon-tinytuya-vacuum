# TARS Home Add-ons

Custom Home Assistant add-ons for the Chatsworth smart home.

## Add-ons

### TinyTuya Vacuum Controller
Local Tuya control for Eufy Robovac S1 Pro (T2080A). Provides a REST API for start/stop/dock/suction control without cloud dependency.

## Installation
1. In HA: Settings → Add-ons → Add-on Store → ⋮ → Repositories
2. Add: `https://github.com/bstanbury/ha-addon-tinytuya-vacuum`
3. Find 'TinyTuya Vacuum Controller' and install
4. Configure device_id, local_key, and device_ip
5. Start the add-on
